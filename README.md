# JUST SAY IT｜瞬間英作文 — 使い方ダイジェスト

「聞いて、口に出すだけ」のかんたん英会話トレーニングアプリ **JUST SAY IT（瞬間英作文）** の
使い方を、はじめての方向けにまとめた縦型（9:16）スライドです。
スライドは自動再生＋ループし、各要素にアニメーションが付いています。

## 📄 ファイル構成

```
.
├── index.html   # 本編（約59秒・全9枚）
├── 30sec.html   # 30秒版（X / SNS向けの短尺）
└── README.md
```

> **`index.html` と `30sec.html` は画像・スクリプトをすべて内部に埋め込んだ自己完結型ファイルです。**
> 外部ファイルを一切参照しないため、**どこに置いても画像が確実に表示されます**
> （別フォルダ・別サーバー不要。ファイル単体で動きます）。

## 🚀 公開（GitHub Pages / Vercel など）

ファイルをそのまま置くだけです。

- **GitHub Pages**: Settings → Pages → Source を `Deploy from a branch`（`main` / `(root)`）に設定
  - 公開URL: `https://<ユーザー名>.github.io/<リポジトリ名>/`
- **Vercel**: リポジトリをインポートするだけ。フレームワーク設定は `Other`（静的）でOK

本編 `…/index.html` ／ 30秒版 `…/30sec.html`

## ⌨️ 操作

- **自動再生**: 開くと自動で進み、最後まで行くと先頭に戻ってループします
- **一時停止 / 再開**: 右上のボタン
- **手動操作**: ← → キー、または画面の左右タップ
